## LawBridge

LawBridge is a full-stack web application designed to connect clients with legal professionals. It provides role-based access for Admin, Lawyers, and Users, enabling efficient legal service management and communication.


##  Features

- Role-based authentication (Admin, Lawyer, User, Guest)
- User management system
- Lawyer listing and interaction
- Admin dashboard for control and monitoring
- Structured frontend-backend architecture
- Responsive UI using modern web technologies
